# vue-html-test-page
Quickly see how your application renders common html elements:
- Headings
- Formatting Elements
- Lists
- Tables
- Forms


### Install
```
$ npm install vue-html-test-page
```

### Usage
```js
import TestPage from 'vue-html-test-page'

Vue.use(TestPage)
```

Then render the test page markup from any Vue file
```html
<html-test-page />
```
